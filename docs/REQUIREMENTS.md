# 레벨2 점심 뭐 먹지?

## 기능 구현 목록

- [x] 레벨 1 점심뭐먹지 미션에서 작성한 마크업을 재사용한다.

- [x] 첫 화면 랜더 시 음식점 목록이 보여야한다.

- [x] 음식점 카테고리별 필터링 기능

  - [x] 카테고리에 해당하는 음식점 리스트를 보여준다.

- [x] 이름순/거리순 정렬 기능

  - [x] 음식점 리스트를 이름 기준으로 정렬한다.

  - [x] 음식점 리스트를 거리 기준으로 정렬한다.

- [ ] 음식점 상세정보 확인 기능
  - [ ] 각 음식점 리스트를 클릭하면, 상세정보 모달을 보여준다.
  - [ ] 상세정보 모달을 닫을 수 있다.
